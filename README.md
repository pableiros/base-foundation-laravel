# base-foundation-laravel

**Composer installation**

`composer.json`

```json
{
    "name": "laravel/laravel",
    ...
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/pableiros/base-foundation-laravel"
        }
    ],
}
```

```bash
composer require pableiros/base-foundation-laravel
```
